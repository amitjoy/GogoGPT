Q: show help for all commands
A: felix:lb

Q: show help for 'echo' command
A: felix:help echo

Q: list contents of a directory
A: felix:ls 'directory'

Q: change directory
A: felix:cd 'directory'

Q: determine from where a bundle loads a class
A: felix:which 'class'

Q: install a bundle using its locations
A: felix:install 'file:/path_to_jar.jar'

Q: uninstall bundle with id 10
A: felix:uninstall 10

Q: refresh bundle with id 10
A: felix:refresh 10

Q: resolve bundle with id 15
A: felix:resolve 15

Q: start bundle with id 20
A: felix:start 20

Q: stop bundle with id 25
A: felix:stop 25

Q: show bundle start level of bundle with id 30
A: felix:bundlelevel 30

Q: set bundle start level to 11 of bundle with id 30
A: felix:bundlelevel 11 30

Q: show start level of framework bundle
A: felix:frameworklevel

Q: change framework start level to 20
A: felix:frameworklevel 20

Q: list bundles matching a substring
A: felix:lb substring

Q: list bundle matching a substring and show locations
A: felix:lb -l substring

Q: list bundles matching a substring and show symbolic names
A: felix:lb -s substring

Q: list bundles matching a substring and show update location
A: felix:lb -u substring

Q: show all capabilities of all bundles
A: felix:inspect cap *

Q: show all capabilities of a bundle having id 1
A: felix:inspect cap * 1

Q: show all packages capabilities of a bundle having id 1
A: felix:inspect cap package 1

Q: show all package capabilities of all bundles
A: felix:inspect cap package

Q: show all service capabilities of a bundle having id 1
A: felix:inspect cap service 1

Q: show all service capabilities of all bundles
A: felix:inspect cap service

Q: show all requirements of all bundles
A: felix:inspect req *

Q: show all requirements of a bundle having id 1
A: felix:inspect req * 1

Q: show all package requirements of a bundle having id 1
A: felix:inspect req *package* 1

Q: show all package requirements of all bundles
A: felix:inspect req package

Q: show all service requirements of a bundle having id 1
A: felix:inspect cap service 1

Q: show all service requirements of all bundles
A: felix:inspect cap service

Q: show session context
A: gogo:set

Q: set a variable into session context
A: variable = value

Q: get value of the variable set in the session context
A: $variable

Q: print the variable set in the session context
A: echo $variable

Q: Match input using regular expression (grep)
A: echo test | grep '[tT]est'

Q: Read the content of a resource from path, for example, a/b/c.txt
A: cat a/b/c.txt

Q: Collect contents of a/b/c.txt into onle line
A: cat a/b/c.txt | tac

Q: Collect contents of a/b/c.txt into a variable, named 'my_var'
A: my_var = (cat a/b/c.txt | tac)
